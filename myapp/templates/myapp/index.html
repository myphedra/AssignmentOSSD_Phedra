SmartDoc AI

AI-powered PDF Q&A · RAG · Local LLM · Open Source Software Development Course

Project 06 View on GitHub

The final project for the Open Source Software Development course — an intelligent, fully local PDF document question-answering system powered by a Retrieval-Augmented Generation (RAG) pipeline. Users can upload any PDF and ask questions in Vietnamese or English; the system retrieves semantically relevant passages and generates grounded answers entirely offline using open-source models.

100% local inference — no cloud APIs, no data transmission. All document content stays on the user's machine.
Semantic Search

FAISS vector database with 768-dim multilingual embeddings for accurate retrieval.

Local LLM

Qwen2.5:7b via Ollama — Vietnamese-optimized with 128K context window.

Bilingual Support

Vietnamese & English with automatic language detection and prompt selection.

50+ Languages

paraphrase-multilingual-mpnet-base-v2 embedding model supports broad multilingual use.

Tech Stack

Django LangChain FAISS HuggingFace Qwen2.5:7b Ollama RAG AWS EC2
